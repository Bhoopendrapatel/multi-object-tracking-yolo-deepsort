# Multi-Object Detection and Tracking using YOLOv8 and DeepSORT

## Overview
This project implements a computer vision pipeline for detecting and tracking multiple objects in a video using YOLOv8 for object detection and DeepSORT for tracking.

## Features
- Real-time object detection using YOLOv8
- Persistent ID assignment using DeepSORT
- Handles multiple objects simultaneously
- Annotated output video with bounding boxes and IDs

## Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8
- DeepSORT

pip install ultralytics opencv-python deep-sort-realtime
